# java數據類型

变量3个基本特征：类型，名称，值
每个变量都占一个存储空间
程序中`+ - * %`的使用

## 基本数据类型

| 数据类型 |                                                   说明                                                    |
| :------: | :-------------------------------------------------------------------------------------------------------: |
|  数值型  | 整数类型，存放整数（byte[1],short[2],int[4],long[8]；浮点（小数类型）（float[4]单精度,double[8]双精度）） |
|  字符型  |                                                  char[2]                                                  |
|  布尔型  |                                         boolean[1],存放true,false                                         |

## 引用数据类型

| 数据类型 |   说明    |
| :------: | :-------: |
|    类    |   class   |
|   接口   | interface |
|   数组   |    []     |


## 整数类型

|     类型      | 占用空间 | 范围  |
| :-----------: | :------: | :---: |
|  `byte`字节   |  1字节   |       |
| `short`短整型 |  2字节   |       |
|   `int`整型   |  4字节   |       |
| `long`长整型  |  8字节   |       |
